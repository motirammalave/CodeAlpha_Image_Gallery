# Image Gallery

This project is a responsive Image Gallery created as part of the CodeAlpha Frontend Development Internship.

## Features
- Responsive grid layout
- Image lightbox preview
- Next and previous image navigation
- Smooth hover effects
- Clean and simple user interface

## Technologies Used
- HTML
- CSS
- JavaScript

## Project Description
The Image Gallery allows users to view images in a grid layout. When an image is clicked, it opens in a lightbox view where users can navigate between images using next and previous buttons.
